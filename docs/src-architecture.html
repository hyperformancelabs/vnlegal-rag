VNLegal RAG — Package src

Pipeline truy hồi văn bản pháp luật tiếng Việt (v1.3), được trích xuất từ notebook thành package Python tái sử dụng được.

Topic-filter (TextCNN) → ứng viên theo macro_domain → chấm điểm sparse (TF-IDF), dense (Siamese LSTM), hoặc hybrid.

21 file Python ~1530 LOC 5 subpackage 4 retrieval modes PyTorch + scikit-learn

1. Tổng quan

src là package đóng gói toàn bộ logic của pipeline truy hồi: nạp dữ liệu & artifacts, dựng chỉ mục, chạy mô hình, và đánh giá. Notebook chỉ điều phối — mọi xử lý nằm trong src. Điểm vào công khai: 

from src import PipelineConfig, RetrievalPipeline, build_markdown_table

pipe = RetrievalPipeline.from_config(PipelineConfig())   # nạp data + artifacts + dựng index
pipe.retrieve_tfidf_textcnn("Hợp đồng lao động thời vụ ...")   # sparse
pipe.retrieve_siamese_textcnn("...")                          # dense  (cần Siamese weights)
pipe.retrieve_hybrid_textcnn("...")                          # hybrid (weighted-sum)
metrics = pipe.evaluate("test")                              # Recall@k + MRR

question→ TextCNN

5. Chi tiết từng thành phần

utils text.py

Tokenizer chuẩn + IO vocab. Không phụ thuộc pyvi/underthesea → chạy mọi nơi.

  • simple_tokenize(text)
    lowercase + regex \w+ (giữ dấu tiếng Việt)
  • encode_text / encode_with_mask
    token→id, pad/truncate; bản mask trả thêm mask 1/0
  • build_vocab / load_vocab / save_vocab
    dựng & đọc tokenizer_vocab.json

utils device.py · io.py

  • default_device()
    CPU/CUDA; override bằng env VNLEGAL_DEVICE
  • read_ready_csv(path)
    đọc CSV tab-separated (split chuẩn)
  • read_json(path)
    đọc JSON UTF-8 (meta/label maps)

data loaders.py

  • locate_repo_root()
    dò repo root từ marker thư mục
  • resolve_data_dir(candidates)
    chọn dir đầu tiên có qa_train/val/test.csv
  • load_corpus / load_qa / load_label_list
    nạp corpus (dedupe passage_id), 3 split QA, danh sách nhãn

data processors.py

  • detect_text_col(df)
    tìm cột text (article_content/text)
  • dedupe_corpus(df)
    bỏ trùng theo passage_id + reset index (căn hàng với ma trận)

models classifier.py

Bộ lọc chủ đề Kim-style TextCNN (filter 3/4/5, max-over-time pool).

  • class TextCNN
    embedding → conv1d → ReLU → maxpool → fc
  • build_textcnn_from_state_dict(state, n)
    suy ra kích thước kiến trúc trực tiếp từ weights

models bi_encoder.py

Dense bi-encoder — masked mean-pool + L2 normalize.

  • class LSTMEncoder
    embedding → LSTM → mean-pool có mask → chuẩn hoá L2
  • class SiameseLSTM
    twin encoder, forward trả cosine similarity
  • infer_siamese_config / add_encoder_prefix_if_needed
    suy config + vá prefix key checkpoint

retrieval sparse.py

  • build_tfidf_index(texts, max_feat)
    fit TF-IDF, trả ma trận L2-normalized
  • tfidf_scores(q, idx, ...)
    cosine TF-IDF query↔corpus[idx]
  • predict_topic_topk(q, textcnn, ...)
    top-k macro_domain từ TextCNN
  • candidate_indices_for_labels(...)
    hợp các index theo nhãn; fallback toàn corpus

retrieval dense.py

  • precompute_doc_embeddings(...)
    encode toàn corpus 1 lần (batch)
  • encode_query_siamese(...)
    encode 1 query → vector L2
  • siamese_scores(q, idx, emb, ...)
    dot-product query↔doc_embeddings[idx]

retrieval fusion.py · hybrid.py

  • minmax_norm(scores)
    scale [0,1] để 2 thang điểm so sánh được
  • weighted_sum(sparse, dense, α)
    α·sparse + (1−α)·dense
  • hybrid_scores(q, idx, ..., α=0.7)
    norm rồi fuse TF-IDF + Siamese trên cùng pool

retrieval __init__.py

  • format_results(df, col, idx, scores)
    dựng DataFrame kết quả xếp hạng
  • top_k_order(scores, k)
    index top-k giảm dần

evaluation metrics.py · evaluator.py

  • rank_of(gold, ranked)
    vị trí 1-based của passage vàng
  • aggregate_metrics(name, ranks, ks)
    gộp thành Recall@k + MRR
  • evaluate_retriever(name, fn, qa, ...)
    chạy retriever trên mẫu QA, thu rank
  • build_markdown_table(rows)
    render bảng markdown

root factory.py

  • select_artifact_dir(candidates, ...)
    chọn dir artifact hợp lệ đầu tiên + file weight
  • load_textcnn(dir, w, device)
    bundle TextCNN: model, stoi, id2label, max_len
  • load_siamese(dir, w, device)
    bundle Siamese; trả None nếu thiếu (degrade)
